📦 You have downloaded: fixitqc-v7.0-deployable.zip

This package contains your complete FixIt QC marketing website
with enhanced admin panel and icon-based navigation.


🚀 INSTALLATION (3 STEPS):

┌─ STEP 1: Extract the ZIP ───────────────────────────────┐
│                                                           │
│  Extract the ZIP file to a folder on your computer.      │
│  You'll see this structure:                              │
│                                                           │
│  fixitqc-site/                                           │
│  ├── index.html          (Homepage)                      │
│  ├── admin.html          (Admin Panel)                   │
│  ├── features.html       (Features Page)                 │
│  ├── pricing.html        (Pricing Page)                  │
│  ├── contact.html        (Contact Form)                  │
│  ├── screenshots.html    (Gallery)                       │
│  ├── login.html          (Admin Login)                   │
│  ├── assets/             (CSS, JS, Images)               │
│  ├── firebase.json       (Hosting Config)                │
│  └── README.md           (Documentation)                 │
│                                                           │
└───────────────────────────────────────────────────────────┘

┌─ STEP 2: Add Your Logo ─────────────────────────────────┐
│                                                           │
│  1. Navigate to: assets/img/                             │
│                                                           │
│  2. Add your logo file as: fixitqc-logo.png             │
│     (The logo with turbine, gear, and oil drop)         │
│                                                           │
│  3. Recommended size: ~300x60px PNG format               │
│                                                           │
└───────────────────────────────────────────────────────────┘

┌─ STEP 3: Deploy ────────────────────────────────────────┐
│                                                           │
│  OPTION A - Firebase Hosting (Recommended):              │
│                                                           │
│    1. Open terminal in the extracted folder              │
│    2. Run: firebase login                                │
│    3. Run: firebase deploy --only hosting                │
│    4. Site live at: fixit-prod-71fcb.web.app            │
│                                                           │
│  OPTION B - Test Locally First:                          │
│                                                           │
│    1. Open terminal in the extracted folder              │
│    2. Run: python3 -m http.server 8000                   │
│    3. Open: http://localhost:8000                        │
│    4. Test all features                                  │
│    5. Then deploy with Firebase                          │
│                                                           │
│  OPTION C - Other Hosting:                               │
│                                                           │
│    Upload all files to any web host that supports        │
│    static sites (Netlify, Vercel, GitHub Pages, etc.)   │
│                                                           │
└───────────────────────────────────────────────────────────┘

🐛 TROUBLESHOOTING:

  Q: Logo not showing?
  A: Make sure file is named exactly: fixitqc-logo.png
     and placed in: assets/img/

  Q: Admin panel blank?
  A: Open browser console (F12) to check for errors
     Make sure all .js and .mjs files are present

  Q: Styles not loading?
  A: Check that assets/css/site.css exists
     Try clearing browser cache (Ctrl+Shift+R)

  Q: Can't deploy to Firebase?
  A: Make sure you've run: firebase login
     Check that .firebaserc and firebase.json exist
